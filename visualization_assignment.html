Line Plot - Cambios de temperatura¶

Este gráfico muestra como cambió la temperatura desde el 4/5/2025 hasta el 10/5/2025. Un gráfico de líneas es ideal para datos temporales porque permite visualizar tendencias y patrones de manera clara.

Elegí esta visualización porque facilita observar aumentos o disminuciones en un intervalo continuo.

In [2]:
days = np.arange(4, 11)
temperature = [23, 19, 22, 20, 19, 20, 20]

plt.figure()
plt.plot(days, temperature, marker='o', linestyle='--', color='blue', label='Temperature')

plt.title('Temperatura 4/5/2025 - 10/5/2025')
plt.xlabel('Días')
plt.ylabel('Temperatura (°C)')

plt.savefig('imgs/temperatura_semana.png', dpi=300, bbox_inches='tight')
plt.show()
